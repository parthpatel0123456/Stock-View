## How To Start

### 1.Activate Venv

source venv/bin/activate

### 2. Run Flask (backend)

python main.py

### 3. Run React (frontend)

cd react-frontend
npm start

### 4. Run Database (SQLite)

sqlite3 /backend/stockview.db
